# Templates

This vault uses two template layers:

- `Templates/QuickAdd/`: structured capture templates driven by QuickAdd.
- `Templates/Templater/`: manual note templates for Templater or the core Templates plugin.

Recommended workflow:

1. Run `QuickAdd: Research Capture` for new raw sources, derived notes, output reports, or question blocks.
2. Use Templater when you want to hand-create a note and still keep the vault's schema consistent.
3. Avoid making manual templates for `wiki/concepts/` or `wiki/sources/`, because those are compiler-owned pages.
